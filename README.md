# AnomaLens

AnomaLens is a full-stack intelligent log anomaly detection and observability platform designed to analyze system logs, detect anomalies using machine learning, and provide actionable operational insights.

## 🚀 Tech Stack

### Backend
- Spring Boot
- Spring Security + JWT
- PostgreSQL
- Spring Data JPA

### ML Service
- FastAPI
- Scikit-learn
- PyTorch
- Pandas / NumPy

### Frontend
- React
- Tailwind CSS
